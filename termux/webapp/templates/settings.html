☁️ XiaoZhi Cloud — MCP Endpoint
Server ini akan dial keluar ke URL di bawah dan mempertahankan koneksi WSS permanen. XiaoZhi cloud mengirim request tools/list dan tools/call lewat koneksi ini. Token JWT bisa diperbarui kapan saja tanpa restart — koneksi akan reconnect otomatis.
Memuat…
URL standar XiaoZhi tidak perlu diubah. Token ditambahkan otomatis sebagai query parameter ?token=...
Token biasanya berformat eyJhbG... (JWT). Berlaku sampai tanggal exp di dalamnya. Kalau expired, perbarui token di sini dan klik Simpan — koneksi akan reconnect otomatis tanpa restart server.

Browser
Dipakai oleh youtube.play_music dan youtube.play_video untuk membuka video secara lokal. Perilaku tergantung platform tempat server ini berjalan:
• Windows/desktop: membuka Brave. Path di bawah biasanya tidak perlu diisi — auto-detect akan mencoba lokasi instalasi umum dulu. Isi manual hanya kalau auto-detect gagal.
• Termux (Android): field path di bawah diabaikan. Video dibuka lewat Android Intent ke browser default sistem (bukan Brave secara spesifik, kecuali Brave sudah diset sebagai default di Android Settings). Butuh paket termux-api + app pendamping Termux:API terinstall.
Kosongkan untuk auto-detect. Tidak berlaku saat server jalan di Termux.
